Node Metadata Properties

Most of these are node properties, though some (noted) are node type properties.
Perhaps we should standardize internal naming, something like:
  fooPath    -- A filename full path.
  fooDir    -- A directory full path.
  fooUri    -- A URI.
  fooName    -- A native name (i.e., node-type specific).

hostRoot hash Subject is $nodeType. Maps the server prefix (such as "http://localhost/") of node URIs of this $nodeType to the root location (as native name) of the server that implements the wrapper for this $nodeType. Analogous to $DOCUMENT_ROOT. $DOCUMENT_ROOT (a/k/a $basePath) is used by default if this property is not set.
